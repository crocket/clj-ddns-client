# clj-ddns-client

It updates DDNS entries.

## Supported DDNS Providers

* dnsever

## What It Doesn't Do.

* It doesn't run as a daemon.

## Installation

* Copy config.sample.edn to config.edn
* Configure config.edn accordingly.

## Usage

```
$ java -jar clj-ddns-client-0.1.2-standalone.jar -h
```
